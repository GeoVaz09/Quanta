Quanta Playground



Welcome to the official playground site.

This is a site for testing Quanta


Steps:

1. Select the script you want to test from the specific examples

2. Press "View Source" to see the exaple's source code in Github (Optional)

3. Press "Run ▶" to copy exaple file's name and go to the Playground (done by the script when "Yes" is clicked)

4. If the automatic copy in the script doesn't work, just see the file names in Github (only the .txt files are examples) and copy the exaple file's name manualy (Optional)

5. After the Playground loads (pythonanywhere.com), on the "File to load:" input, paste the exaple file's name

3. Press run and enter the number of the example script you want to load (after pythonanywhere.com loads)

4. Use the script as instructed by the tutorial (Just see the tutorial first)

5. For any questions or ideas, just send me on Slack @George Vazakas
